Botões

Botões customizados com múltiplos estilos, tamanhos e estados.

Exemplo Básico

Use qualquer um dos estilos de botões disponíveis.

btn btn-danger">Danger
HTML 
<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>
<button class="btn btn-danger">Danger</button>
<button class="btn btn-warning">Warning</button>
<button class="btn btn-info">Info</button>
<button class="btn btn-light">Light</button>
<button class="btn btn-dark">Dark</button>

Botões com Ícones

Botões com ícones usando Bootstrap Icons.

HTML 
<button class="btn btn-primary">
    <i class="bi bi-plus-lg"></i>
    Adicionar
</button>

<button class="btn btn-success">
    <i class="bi bi-check-lg"></i>
    Confirmar
</button>

<button class="btn btn-danger">
    <i class="bi bi-trash"></i>
    Excluir
</button>

<button class="btn btn-outline-primary">
    <i class="bi bi-pencil"></i>
    Editar
</button>

Grupo de Botões

Agrupe botões emtoolbar ou grupos.

Toolbar
Group
Vertical
HTML 
<!-- Toolbar -->
<div class="btn-toolbar" role="toolbar">
    <div class="btn-group me-2" role="group">
        <button class="btn btn-secondary"><i class="bi bi-format-bold"></i></button>
        <button class="btn btn-secondary"><i class="bi bi-type-italic"></i></button>
    </div>
    <div class="btn-group" role="group">
        <button class="btn btn-primary"><i class="bi bi-save"></i></button>
    </div>
</div>

<!-- Group -->
<div class="btn-group" role="group">
    <button class="btn btn-outline-primary">Esquerda</button>
    <button class="btn btn-outline-primary active">Centro</button>
    <button class="btn btn-outline-primary">Direita</button>
</div>

<!-- Vertical -->
<div class="btn-group-vertical" role="group">
    <button class="btn btn-secondary"><i class="bi bi-house"></i></button>
    <button class="btn btn-secondary"><i class="bi bi-person"></i></button>
    <button class="btn btn-secondary"><i class="bi bi-gear"></i></button>
</div>
