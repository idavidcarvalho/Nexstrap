Nexstrap

Design System moderno baseado em Bootstrap 5. Componentes customizados, acessíveis e altamente personalizáveis.

Documentação GitHub

Personalizável

Variáveis CSS para fácil customização. Altere cores, fontes e espaçamento em um único lugar.

Acessível

Suporte completo a WCAG 2.1. ARIA labels, focus states e navegação por teclado.

Responsivo

Layout adaptável para todos os dispositivos. Mobile-first e breakpoints do Bootstrap.

Código Limpo

CSS e JS externos sempre. Sem inline styles ou scripts.

Tema Escuro

Suporte nativo a tema escuro. Alternância automática ou manual.

Open Source

Licença MIT. Livre para usar em projetos pessoais e comerciais.

Quick Start

Adicione o Nexstrap ao seu projeto em poucos minutos.

Via CDN

HTML 
<!-- Nexstrap CSS -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/nexstrap/dist/css/nexstrap.min.css">

<!-- Bootstrap Icons (opcional) -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css">

<!-- Nexstrap JS -->
<script src="https://cdn.jsdelivr.net/npm/nexstrap/dist/js/nexstrap.min.js"></script>

Via NPM

Bash 
npm install nexstrap
