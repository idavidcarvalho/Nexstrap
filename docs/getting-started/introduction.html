Introdução

Nextrap é um Design System moderno baseado em Bootstrap 5, com alta personalização visual e componentes acessíveis.

O que é o Nexstrap?

Nextrap é um sistema de design que estende o Bootstrap 5 com componentes customizados, variáveis CSS para fácil personalização e suporte completo a acessibilidade (WCAG 2.1).

Nota: Nexstrap é um tema/extensão do Bootstrap 5, não um fork. Você precisa do Bootstrap 5 instalado para usar.

Características

  • Variáveis CSS para customização fácil
  • Componentes adicionais (Stat Cards, Profile Cards, etc)
  • Suporte a tema escuro nativo
  • Acessibilidade completa (WCAG 2.1)
  • Layout responsivo mobile-first
  • Código limpo (CSS/JS externos)
  • Documentação completa com exemplos

Quick Start

Adicione o Nexstrap ao seu projeto em poucos minutos:

HTML 
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Projeto</title>
    
    <!-- Bootstrap 5 CSS (OBRIGATÓRIO) -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
    
    <!-- Nextrap CSS -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/nexstrap/dist/css/nexstrap.css">
    
    <!-- Bootstrap Icons (opcional) -->
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css">
</head>
<body>
    <h1>Olá, Nexstrap!</h1>
    
    <!-- Bootstrap 5 JS (OBRIGATÓRIO) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
    
    <!-- Nexstrap JS -->
    <script src="https://cdn.jsdelivr.net/npm/nexstrap/dist/js/nexstrap.js"></script>
</body>
</html>

Recursos Obrigatórios

Para usar o Nexstrap corretamente, você precisa incluir:

Recurso Obrigatório Descrição
Bootstrap 5 CSS Sim Framework CSS base
Bootstrap 5 JS Sim Para componentes interativos
Nexstrap CSS Sim Estilos customizados
Nexstrap JS Recomendado Scripts customizados (toast, sidebar, etc)
Bootstrap Icons Opcional Biblioteca de ícones
